📊 智能學生成績統計分析系統 使用說明

哈囉！歡迎使用學生成績統計分析系統。本系統專為同學們設計，能幫你將多次段考、模擬考等成績，轉換為直觀的能力雷達圖、成績趨勢折線圖，並針對你的強勢與弱勢學科，提供專屬的個人讀書策略建議。所有的資料都只會儲存在你自己的手機或電腦瀏覽器中，不會外流，請放心操作！

系統線上版網址：
https://froglssh.github.io/student-grades-analyzer-online/

🔑 步驟一：查詢與準備你的成績

📱 成績查詢方式：
在開始輸入前，請先透過以下方式查出你想要分析的段考或模擬考成績：
  1. 使用手機登入「酷課 APP」，點選成績查詢。
  2. 或者使用手機/電腦登入學校的「校務行政系統」，查詢各科成績與班級、類組排名。
  3. 將成績記下來，準備手動輸入到本系統中。

二、輸入基本資料與成績

  1. 填寫基本資料： 在「學生基本資料」區，輸入你的姓名、班級、座號與學年度。
  2. 新增考試記錄：
    • 系統預設有一筆考試欄位。若要輸入多次考試（例如：第一次段考、第二次段考、模擬考），點選下方綠色的 ➕ 新增考試記錄 按鈕。
    • 選擇本次考試的類型，並輸入你的「班級排名」與「類組排名」（若有）。
  3. 手動輸入各科成績：
    • 在各個科目格子中，手動輸入你剛才查到的成績分數。
    • 系統預設包含國文、英文、數學、物理、化學、生物、地科、歷史、地理、公民等科目。
    • 增減科目： 若你的考試科目與預設不同，可以點擊科目格子右上角的 × 移除；若有其他科目（如：資訊科技、美術），可點擊 ➕ 新增科目 按鈕自行輸入新增。
  4. 開始分析： 輸入完成後，點選藍色的 📈 開始分析 按鈕，系統就會立即為你生成分析報告。

三、如何看懂分析報告？

點選「開始分析」後，下方會出現多個分頁，點擊分頁標籤可切換檢視：

  • 📊 總覽：
    • 能力雷達圖： 越往外擴展代表分數越高，方便一眼看出你的強科與弱科。
    • 各科平均成績與分布： 讓你了解自己各學科的整體表現。
  • 🔍 單次考試分析： 選擇特定一次的段考，檢視該次考試的所有學科詳細統計與表現。
  • 📈 多次成績分析：
    • 當你輸入兩次以上的考試後，這裡會繪製出「各科成績變化趨勢折線圖」以及「排名變化曲線圖」。
    • 線條向上代表成績進步，向下的話則需要多加注意囉！這能幫你追蹤自己的成長軌跡。
  • 🎯 優劣勢與策略：
    • 系統會自動將你的科目分為「強勢學科」與「待提升學科」。
    • 針對不同特性的學科，系統會給予你「具體的讀書策略與時間分配建議」，例如如何分配複習進度、如何調整學習方法等。

四、資料保存與備份

為了保護隱私，本系統不會上傳任何資料到伺服器，資料只保存在你目前使用的瀏覽器中。如果你換了手機或清除瀏覽器快取，資料可能會消失，因此請善用以下功能：

  • 💾 儲存資料： 點擊後，資料會保存在此瀏覽器的 Local Storage 中，下次用同一個瀏覽器打開會自動載入。
  • 📂 載入資料： 手動載入你之前在該瀏覽器儲存的成績。
  • 📤 匯出原始資料： 點擊後會下載一個 .json 檔案，這是你的成績備份檔，請妥善保存。
  • 📥 匯入原始資料： 當你換了新手機或瀏覽器時，可點選此按鈕並選取你的 JSON 備份檔，成績便會立刻全部還原！
  • 📄 匯出完整 PDF 報告： 點選此按鈕可以將你美美的成績分析圖表與策略建議，列印出來或下載存成 PDF，方便拿給家長、老師或自己收藏！

五、常見問題 FAQ

1. 為什麼我的雷達圖看起來圓圓的沒有起伏？

可能是因為你所有科目輸入的分數都非常接近。通常各科分數有高低起伏時，雷達圖就會呈現明顯的多邊形，指出你的優勢方向。

2. 怎麼移除已經新增的考試？

在每次考試記錄欄位的右下角，點擊紅色的 🗑️ 移除此考試 即可刪除該次記錄。

3. 輸入成績時，如果缺考該怎麼辦？

如果該科缺考，可以直接將該格分數留空，系統在分析時便會自動忽略該科，不會影響其他科目的平均計算。
