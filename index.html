👥 兩種使用模式說明

本程式同時支援「學生自主查詢輸入」與「教師批次班級分析」。請依據您的角色參考下方對應的引導指南：

學生端 自主學習與策略診斷

1. 查詢個人成績：

  • 使用手機登入 「酷課 APP」 查詢各次段考成績。
  • 或使用手機/電腦登入學校的 「校務行政系統」，記錄每次考試的「各科得分」、「班級排名」與「類組排名」（若有）。

2. 輸入與分析：

  • 在「基本資料」填入姓名、座號。
  • 點擊 ➕ 新增考試記錄 手動輸入各考科分數。
  • 輸入完後，點擊藍色 📈 開始分析 按鈕，即刻觀看「能力雷達圖」與「讀書時間分配建議」。

教師端 班級成績整合與一鍵選生

1. 下載班級考卷 Excel 檔：

  • 登入 「校務行政系統」 的成績管理模組。
  • 依據考試（如：一小段、二小段、期末考），下載班級的成績冊 Excel 原始檔。

2. 批次匯入與自動分析：

  • 在網頁頂端點選 📂 匯入全班成績 Excel，可一次框選多個下載的 Excel 成績檔上傳。
  • 上傳後系統會**自動在背景完成分析與班級名單整合**。
  • 點選 選取特定學生 可直接展開全班名單（含歷次平均），點選名單中的任一位學生即可直接在主畫面顯示該生的個人分析圖表，無須手動逐一輸入！

💾 成績數據保存與匯出按鈕功能大解析

為了確保隱私，本系統將所有資料安全地存放在您當前所使用瀏覽器的 Local Storage 中。當您清除瀏覽器快取、換電腦或換手機時，資料可能遺失。因此，系統提供了多種備份與匯出的按鈕，請詳閱其功能差異：

功能按鈕 操作對象 資料儲存位置 / 格式 適用對象 適用情境
💾 儲存資料 單一學生 瀏覽器 Local Storage 學生、教師 手動修改或輸入某位學生的成績後，點擊進行瀏覽器端暫存，以防關閉網頁後遺失。
📂 載入資料 單一學生 從瀏覽器 Local Storage 讀取 學生、教師 手動開啟網頁時，自動或手動還原先前點擊「儲存資料」的該名學生的成績。
📤 匯出成績 JSON 全系統資料 下載為 JSON 檔案 (.json) 學生、教師 完整資料備份。下載包含「系統所有儲存的學生」以及「已匯入的歷次全班 Excel 原始考卷資料」，用於更換電腦或瀏覽器時備份資料庫。
📥 匯入成績 JSON 全系統資料 從 JSON 檔案 (.json) 載入 學生、教師 將之前備份的 JSON 檔重新上傳，完整還原所有儲存的學生名單與全班考卷原始數據。
📤 匯出全部成績 全班合併資料 下載為 Excel 試算表 (.xlsx) 教師 班級成績大總表。將匯入的歷次全班考卷整合，依「班級➔座號➔考試順序」排序，並在不同學生間插入空白列、凍結首行與姓名/考試欄位，供列印或期末總成績統計。
📥 匯出 CSV 特定學生 下載為 CSV 檔案 (.csv) 學生、教師 僅匯出被選取之特定單一學生的歷次考科成績。適用於單一學生的學業歷程歸檔或匯入其他統計分析軟體。
⚠️ 備份重要提醒：
若您是教師，建議在每次批次匯入全班 Excel 考卷並完成自動分析後，點擊 「📤 匯出成績 JSON」 下載備份。下次使用時，只需點擊 「📥 匯入成績 JSON」，即可立刻召回全班所有學生的所有歷次成績紀錄，不需每次都重新尋找並匯入多張原始 Excel 考卷！

❓ 常見問題 FAQ

Q1. 匯出全部成績的 Excel，為什麼左右滾動時有些欄位不會動？

這是本系統專為教師設計的凍結窗格（Freeze Panes）功能。當您在 Excel 中左右滑動表格時，最左側的 6 個欄位（學年、學期、班級、座號、姓名、考試）會固定在左側，不會被滑出螢幕之外。這能確保您滑到最右側看各個考科的成績時，依然能對齊這是哪一位學生、哪一次考試的成績！

Q2. 手機開啟「酷課 APP」或「校務行政系統」查到的成績可以批次匯入嗎？

不行，由於手機端 APP 主要是提供家長與學生查詢，沒有直接匯出整班 Excel 的功能。若是學生，請手動抄錄成績後手動在網頁輸入；若是教師，請登入電腦版校務行政系統的「成績冊下載」功能，下載標準的班級 Excel 檔案後，再使用批次匯入功能。

Q3. 匯入全班 Excel 時，可以同時選擇很多個檔案嗎？

可以的！在點選「匯入全班成績 Excel」按鈕並開啟檔案選取視窗時，您可以按住鍵盤的 Ctrl 鍵（Mac 請按 Command ⌘ 鍵）並點選多個 Excel 成績檔（例如：第一次段考、第二次段考、期末考），即可一次完成批次匯入，系統會自動將同一個學生的多次成績上下對齊合併。
